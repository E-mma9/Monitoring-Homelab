# Grafana Dashboards

Export your dashboards from Grafana and save the JSON files here.

## How to export

1. Open a dashboard in Grafana
2. Click the gear icon (Dashboard settings)
3. Click **JSON Model** (left side)
4. Copy the entire JSON
5. Save as `cluster-overview.json` or `alerts-overview.json`

## Dashboards in this directory

- `cluster-overview.json` — CPU, RAM, disk, network gauges for all nodes
- `alerts-overview.json` — Active alerts table and alert timeline
